# Project Scope — OpenCart BA Simulation Project

## Project Overview
This project is a Business Analyst simulation conducted on the 
OpenCart demo e-commerce platform (demo.opencart.com). The goal 
is to analyze user workflows, document requirements, design test 
cases, and track defects following Agile methodology — simulating 
real-world BA responsibilities within a software development team.

## Objectives
- Identify and document functional and non-functional requirements 
  for key e-commerce workflows
- Author user stories with clearly defined acceptance criteria
- Design a comprehensive test plan and execute manual test cases
- Log and track defects with reproducible steps and severity ratings
- Organize all work within a simulated Agile sprint structure

## Out of Scope
- Backend/database analysis
- Admin panel functionality
- Payment gateway integration
- Mobile application testing
- Performance or load testing

## Assumptions
- Testing is conducted on the publicly available demo site
- Data entered during testing is temporary and may be reset by the site
- All findings reflect the demo environment, not a production system
