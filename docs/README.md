# OpenCart BA Simulation Project

A Business Analyst simulation project analyzing the OpenCart 
demo e-commerce platform. Covers end-to-end BA deliverables 
including requirements documentation, user stories, test 
planning, manual test execution, and defect tracking — 
all following Agile methodology.

## Modules Under Analysis
- User Registration & Login
- Product Search & Filtering
- Shopping Cart & Checkout
- Order Management

## Tools & Methodology
- **Site tested:** https://demo.opencart.com
- **Methodology:** Agile/Scrum — 2 simulated sprints
- **Documentation:** Markdown
- **Version control:** GitHub
